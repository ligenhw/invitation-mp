## Introduction 介绍 ✨

invitation-mp 是一款开源免费的婚礼邀请函 小程序。

小程序名称 : 欢聚一刻

小程序简介 : 欢聚一刻是一款开源免费的邀请函服务。让大家轻松制作精美的婚礼邀请函，聚会邀请函。
您的问题和建议，通过邮件ligenhw@outlook.com反馈。


## 功能 🔥

* 实时弹幕
* 婚礼海报
* 相册
* 导航
* 祝福留言
* 我的
* 邀请函制作
* 礼金管理
* 视频点播
